# Flower Application

This folder contains the runnable federated learning app. See the root `README.md` for setup, data preparation, and run commands.

```text
flower-fl/
├── client/       # FL clients and local client data
├── common/       # Shared CNN, data loading, and metrics helpers
├── server/       # Flower server entry point and server dependencies
└── tools/        # Simulation and network diagnostic utilities
```

Keep experiment logs, reports, generated outputs, and legacy notebooks outside this folder so the runtime app stays focused.
